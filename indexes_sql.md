```
CREATE INDEX idx_team_member_user_team 
ON team_member(user_id, team_id);
```

Этот индекс нужен для быстрой проверки доступа пользователя к команде. Он ускоряет запросы вида WHERE user_id = ? AND team_id = ?. Без него PostgreSQL будет просматривать всю таблицу team_member, что становится критично при большом количестве участников. Это один из самых частых запросов в системе.
